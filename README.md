# Online Courses — Test Task

Карточный каталог онлайн-курсов с поиском и фильтрацией по категориям.

## Стек

- HTML5
- SCSS → CSS (скомпилирован через `sass`)
- JavaScript (vanilla, без библиотек)

## Функциональность

- Pixel-perfect вёрстка по макету Figma (1920px → 320px)
- Живой поиск по названию курса (без перезагрузки)
- Фильтрация по категориям
- Кнопка «Load more» с пагинацией
- Адаптивная верстка под все размеры экранов
- BEM-методология

## Запуск

Открыть `index.html` в браузере — никаких зависимостей для работы не нужно.

Для пересборки стилей:

```bash
npm install
npm run build
```

## Время выполнения

Около 2 часов.
